## Azure data lake storage
存储大容量数据


✅ 第4步：处理数据（ETL）
- 你可以用 Auto Loader 实时导入文件，也能跑 Spark job 把原始数据变干净数据表。

✅ 第5步：训练机器学习模型（MLflow）
用 sklearn、PyTorch、XGBoost 等库训练 → MLflow 自动记录实验 → 一键部署服务

✅ 第6步：对接其他 Azure 服务
- 存储在 ADLS
- 分析结果发送给 Power BI
- 调用 Azure OpenAI 进行内容生成
