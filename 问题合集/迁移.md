7. SLA，SLO，SLI是什么？

    **答案**：
    - **SLA（Service Level Agreement）**：合同/协议，写进法律文件里的承诺，通常是厂商对客户的。
    - **SLO（Service Level Objective）**：目标值，是架构师/运维团队设定的期望，比如 API 成功率 ≥ 99.9%。
    - **SLI（Service Level Indicator）**：实际指标，监控采集到的数据，比如“8月API成功率 = 99.93%”。
